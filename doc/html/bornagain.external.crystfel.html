bornagain.external.crystfel module¶

Utilities for working with CrystFEL files. Note that you can convert a CrystFEL geometry file to a Python dictionary object with the cfelpyutils.crystfel_utils.load_crystfel_geometry() function. Most of the functions below wrap around the cfelpyutils package.

bornagain.external.crystfel.load_crystfel_geometry(geometry_file)[source]¶

Given a CrystFEL geometry file, create a python dictionary object. This uses the cfelpyutils module - blame Valerio Mariani if it’s broken :)

Parameters:geometry_file (str) – Path to geometry file
Returns:Dict

Given a CrystFEL geometry dictionary, create a list of :class:<bornagain.geometry.PADGeometry objects. This will also append the name of the panel to the PADGeometry instance.
