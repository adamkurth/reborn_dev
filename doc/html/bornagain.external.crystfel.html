bornagain.external.crystfel module¶

Method for working with CrystFEL geometry files. Note that you can convert a CrystFEL geometry file to a Python dictionary object with the cfelpyutils.crystfel_utils.load_crystfel_geometry() function.

bornagain.external.crystfel.geometry_dict_to_pad_geometry_list(geometry_dict)[source]¶

Given a CrystFEL geometry dictionary, create a list of :class:<bornagain.geometry.PADGeometry` objects. This will also append the name of the panel to the PADGeometry instance.

Parameters:geometry_file (str) – Path to geometry file

Returns: List of PADGeometry instances

bornagain.external.crystfel.geometry_file_to_pad_geometry_list(geometry_file)[source]¶

Given a CrystFEL geometry file, create a list of PADGeometry objects. This will also append extra crystfel-specific items like fsx, max_fs, etc.

Parameters:geometry_file (str) – Path to geometry file

Returns: List of PADGeometry instances

  • geom_dict (dict) – Geometry dictionary
Returns:
